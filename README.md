# Gym Management Database System

## Project Description
## Database Creation

The database schema was designed using MySQL Workbench and exported using the Data Export feature.
The `create_tables.sql` file contains the full MySQL dump including table definitions,
constraints, and engine configurations.
This project is a multi-branch gym management database system designed as part of a graduate-level course.
